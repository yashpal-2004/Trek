# Garhwal Expedition Planner

Premium travel planning web application for a 10-day budget backpacking trip across Garhwal, Uttarakhand.

## Quick Start

```bash
cd garhwal-expedition
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Build & Deploy

```bash
npm run build
npm run preview
```

Deploy the `dist/` folder to Vercel, Netlify, or any static host.

## Features

- Complete 10-day itinerary with accordion details
- Animated route timeline & statistics
- Transport guide (22+ routes)
- Trek guides (6 treks)
- Budget calculator with Recharts visualizations
- Expense tracker (LocalStorage)
- Interactive packing checklist
- Stay, food, safety & emergency guides
- Photo gallery with lightbox
- FAQ with search & categories
- Print & JSON export
- Scroll spy navigation
- Fully responsive (320px–1920px+)

## Tech Stack

React · Vite · Tailwind CSS · Framer Motion · Recharts · React Hook Form · Lucide React
